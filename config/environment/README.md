# Environment Configuration

## Purpose

Configuration guides for environment setup, API keys, and external service integration.

## Guides

- **`SETUP_GROQ.md`**: Groq API key setup instructions
- **`GET_GROQ_KEY.md`**: How to obtain Groq API key
- **`EMAIL_TROUBLESHOOTING.md`**: Email configuration troubleshooting

## Required Configuration

### Groq API Key

Required for AI evaluation features. See `GET_GROQ_KEY.md` for instructions.

### Optional Configuration

- Email settings (see `EMAIL_TROUBLESHOOTING.md`)
- Blockchain network configuration
- API rate limiting

## Usage

Set environment variables in `.env` file or export in shell:

```bash
export GROQ_API_KEY=your-key-here
```

## Documentation

See individual guide files for detailed setup instructions.
