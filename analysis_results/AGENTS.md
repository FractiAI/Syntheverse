# Embedding Analysis Results

## Purpose

Output directory for RAG embedding analysis, visualization, and validation results generated by the analysis pipeline in `src/api/rag_api/analysis/`.

## Contents

### Visualization Outputs

- **`embedding_distributions.png`**: Distribution histograms of embedding values and norms
- **`pca_2d_scatter.png`**: 2D PCA scatter plot of embeddings
- **`pca_3d_scatter.png`**: 3D PCA scatter plot of embeddings
- **`pca_variance_explained.png`**: Variance explained by principal components
- **`similarity_heatmap.png`**: Pairwise similarity heatmap
- **`statistics_dashboard.png`**: Combined statistics visualization

### Analysis Reports

- **`similarity_analysis.json`**: Pairwise similarity metrics and duplicate detection
- **`statistics.json`**: Comprehensive embedding statistics (mean, std, norms)
- **`validation_report.json`**: Embedding quality validation results

## Integration Points

- Generated by `src/api/rag_api/analysis/` CLI tools
- Reads embeddings from `src/data/vectorized/embeddings/`
- Visualization uses matplotlib and seaborn
- Statistics computed via numpy and scipy

## Generation Commands

```bash
# Generate all analysis outputs
python -m src.api.rag_api.analysis.cli.analyze_embeddings \
    --embeddings-dir ./src/data/vectorized/embeddings \
    --output-dir ./analysis_results

# Generate visualizations
python -m src.api.rag_api.analysis.cli.visualize_embeddings \
    --embeddings-dir ./src/data/vectorized/embeddings \
    --plot-type all --format png

# Validate embeddings
python -m src.api.rag_api.analysis.cli.validate_embeddings \
    --embeddings-dir ./src/data/vectorized/embeddings \
    --output-file ./analysis_results/validation_report.json
```
