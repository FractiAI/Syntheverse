# Embedding Analysis Results

This directory contains analysis results and visualizations from the Syntheverse RAG embedding analysis pipeline.

## Directory Contents

### Visualization Files
- `embedding_distributions.png` - Embedding value distributions
- `pca_2d_scatter.png` - 2D PCA scatter plot
- `pca_3d_scatter.png` - 3D PCA scatter plot
- `pca_variance_explained.png` - PCA variance analysis
- `similarity_heatmap.png` - Embedding similarity heatmap
- `statistics_dashboard.png` - Comprehensive statistics visualization

### Data Files
- `similarity_analysis.json` - Detailed similarity analysis results
- `statistics.json` - Embedding statistics and metrics
- `validation_report.json` - Quality validation results

## Analysis Pipeline

The analysis results are generated by the RAG embedding analysis CLI tools:

1. **Data Loading**: Load embeddings from vectorized directory
2. **Statistics Calculation**: Compute distributions, norms, and quality metrics
3. **Dimensionality Reduction**: Apply PCA for visualization
4. **Similarity Analysis**: Calculate pairwise similarities and clustering
5. **Validation**: Quality checks and outlier detection
6. **Visualization**: Generate plots and dashboards

## Usage

```bash
# Generate comprehensive analysis
python -m src.api.rag_api.analysis.cli.analyze_embeddings \
    --embeddings-dir ./src/data/vectorized/embeddings \
    --output-dir ./analysis_results

# Create visualizations only
python -m src.api.rag_api.analysis.cli.visualize_embeddings \
    --embeddings-dir ./src/data/vectorized/embeddings \
    --plot-type all
```
