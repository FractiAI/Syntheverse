# Data Directory

This directory contains the data files generated by the RAG pipeline.

## Structure

```
data/
├── pdfs/              # Downloaded PDF files from Zenodo
├── parsed/            # Parsed PDF chunks (JSON files)
└── vectorized/        # Vectorized embeddings
    ├── embeddings/    # Embedding JSON files (one per PDF)
    └── metadata/      # Vectorization metadata
```

## Usage

These directories are automatically created and populated when running:

1. **Scraper**: Downloads PDFs to `pdfs/`
2. **Parser**: Saves parsed chunks to `parsed/`
3. **Vectorizer**: Saves embeddings to `vectorized/embeddings/`

## Notes

- These directories are in `.gitignore` (data files are not versioned)
- The RAG API reads from `vectorized/embeddings/` for queries
- You can specify custom paths in the script arguments
