# Metadata Directory

This directory contains metadata files from scraping and processing operations.

## Files

- `test_scrape_results.json`: Test scraping results
- `zenodo_scrape_results.json`: Zenodo repository scraping results
- `scrape_results.json`: Main scraping results (in parent directory)

## Usage

These files are generated automatically when:
- Running the PDF scraper
- Processing Zenodo repositories
- Running test scraping operations
