# Examples

This directory contains example scripts and demonstrations for the Syntheverse PoC system.

## Files

- `demo_poc_system.py` - Complete demonstration of the PoC evaluation workflow
- `demo_interface.html` - Static HTML demonstration of the full collaborator UI
- `test_ui.html` - Test interface for UI components

## Running Examples

### Python Demo
```bash
python demo_poc_system.py
```

### HTML Demos
Open the HTML files directly in a web browser to see static demonstrations of the interfaces.

## Note

These examples are for demonstration purposes and may require the full Syntheverse system to be running for full functionality.
