# API Documentation Agents

## Purpose

API documentation for the Syntheverse system, covering both the RAG API and PoC API with endpoint specifications, integration examples, and usage guides.

## Key Modules

### RAG API Documentation (`RAG_API.md`)

- **RAG API Overview**: REST API for querying the Syntheverse RAG system
- **Features**: Local retrieval, LLM integration, REST endpoints, web UI, CORS enabled
- **Endpoints**: Query, search, health check, statistics
- **Usage Examples**: Python, cURL, JavaScript integration
- **Configuration**: Embedding models, LLM providers (Groq, Ollama, HuggingFace)
- **Performance**: Query processing times, memory usage, cost considerations
- **Troubleshooting**: Common issues and solutions

### PoC API Documentation (`README.md`)

- **PoC API Overview**: REST API for contribution management
- **Contribution Management**: Submit, evaluate, archive, statistics endpoints
- **Sandbox Map**: Visualization data generation
- **Tokenomics**: Epoch information and statistics
- **Blockchain Integration**: Certificate registration
- **Integration Examples**: Frontend integration patterns

## Integration Points

- API documentation references code components in `src/api/`
- Endpoint documentation describes actual API implementations
- Integration examples show frontend-backend connections
- Configuration guides reference environment setup files

## Development Guidelines

- Document all public API endpoints with request/response formats
- Include code examples for common integration patterns
- Update documentation when API endpoints change
- Provide troubleshooting guides for common issues
- Reference configuration files for setup requirements

## Common Patterns

- RESTful API endpoint documentation
- JSON request/response format examples
- Multi-language integration examples (Python, JavaScript, cURL)
- Configuration and troubleshooting sections
- Performance and scaling considerations
