# Contributor Resources

This directory contains resources for contributors to the Syntheverse ecosystem.

## SYNTH Pitch

See `SYNTH_Pitch.md` for the complete contributor pitch document, including:

- Ecosystem introduction and achievements
- Contribution opportunities (Gold/Silver/Copper tiers)
- Token-based reward system
- Benefits and getting started guide

## Contribution Opportunities

### Gold Contributors (Scientific)
- Research and foundational discovery
- Priority access to AI tools
- Influence over scientific directions
- Enhanced PoC/PoD/PoT submission impact

### Silver Contributors (Technological)
- Tool and system implementation
- Early access to development modules
- Contribution-weighted prestige
- Mid-level routing and analysis priority

### Copper Contributors (Alignment)
- Governance and ethical alignment
- Symbolic and operational consistency
- Baseline access to submissions and tools

### Financial Contributors
- Funding for grants and development
- Recognition through influence and early access

## Getting Started

1. Review the SYNTH pitch document
2. Join the community and explore contribution opportunities
3. Start with small contributions to build reputation
4. Earn ERC-20 internal tokens for ecosystem influence
